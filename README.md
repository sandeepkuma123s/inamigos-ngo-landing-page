# 🌍 InAmigos NGO Awareness Webpage

A responsive NGO awareness website developed to showcase the mission, projects, and social impact of **InAmigos Foundation**. This project aims to spread awareness about the organization's initiatives and encourage people to volunteer, support, and contribute to social causes.

## 🚀 Live Demo

🔗 Live Website:
https://sandeepkuma123s.github.io/inamigos-ngo-landing-page/

🔗 GitHub Repository:
https://github.com/sandeepkuma123s/inamigos-ngo-landing-page

---

## 📌 Features

* Responsive and modern user interface
* Introduction to InAmigos Foundation
* Information about ongoing projects
* Social impact and mission section
* Gallery and visual content
* Call-to-action section for volunteers and supporters
* Mobile-friendly design

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* Responsive Web Design
